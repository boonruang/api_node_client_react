# API with nodejs

## initial project

```sh
npm init -y
```

## create server.js

```sh
npm i express bcryptjs jsonwebtoken config express-validator mongoose
```

## Install nodemon concuurently

```sh
npm i -D nodemon concurrently

```

## Git commit

```sh
add .gitignore
git init
git add .
git commit -m 'Express server set with inital routes'
```
